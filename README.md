# Amal Krishna | Software Engineer (MERN Stack)

A modern, responsive, and premium personal portfolio website showcasing my identity as a self-taught software engineer and tech enthusiast.

## 🚀 Live Demo
You can view the repository and host it via GitHub Pages:
[https://amalkrishnaks.github.io/Amal-Krishna-personal-portfolio/](https://amalkrishnaks.github.io/Amal-Krishna-personal-portfolio/)

## ✨ Features
- **Modern UI/UX**: Dark-themed, sleek design with premium aesthetics and smooth animations.
- **Hero Section**: High-impact introduction with a dynamic typing effect.
- **About Me**: Detailed bio highlighting my journey as a self-taught developer.
- **Skills & Expertise**: Visualization of proficiency in the MERN stack (MongoDB, Express, React, Node.js).
- **Projects**: Grid-based showcase of featured development projects.
- **Services**: Overview of professional offerings like Web & API Development.
- **Contact Form**: Integrated with Google Forms for direct communication, featuring a custom confirmation popup.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop views.

## 🛠️ Tech Stack
- **Structure**: Semantic HTML5
- **Styling**: Vanilla CSS3 (Custom Design System)
- **Interactivity**: Modern JavaScript (ES6+)
- **Integration**: Google Forms API (Submissions)
- **Icons**: Font Awesome 6.0

## 📂 Project Structure
```text
├── index.html    # Main HTML structure
├── style.css     # Custom CSS styles
├── script.js     # Interactivity logic
└── README.md     # Project documentation
```

## 🛠️ Setup Locally
To run this project on your local machine:
1. Clone the repository:
   ```bash
   git clone https://github.com/amalkrishnaks/Amal-Krishna-personal-portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Amal-Krishna-personal-portfolio
   ```
3. Open `index.html` in your favorite browser.
